Assurez-vous qu'un server MySQL est actif en local et est disponible sur le 3306 (port par défaut)[localhost:3306].

Pour lancer le programme, il faut créer et activer un environnement virtuel. Puis à la racine du projet
taper les commandes suivantes:

	1. Installer les requirements: 	# pip install -r requirements.txt
	2. Lancer le projet: 		# python main.py

NB: Par défaut, une base de donnée cngdatabase est créée de même qu'un utilisateur avec le couple ('root', 'root')
pour la page de connexion.
